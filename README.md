# Early Lung Cancer Detection Using TCGA-LUAD Gene Expression Data

## Project Overview

This project is a final project for the **Computational Biology** course, Computer Science Program, BINUS Alam Sutera, Semester 4.

The project focuses on building a computational biology workflow for **early lung cancer detection** using **gene expression data** and **machine learning**. The main objective is to classify TCGA-LUAD samples into **Tumor** or **Normal** based only on RNA-seq gene expression values.

This project does not aim to build a clinical diagnostic system. Instead, it demonstrates how biological/genomic data can be processed and analyzed using computational methods.

## Topic

**Early Lung Cancer Detection Using TCGA-LUAD Gene Expression Data and Machine Learning**

The project uses gene expression data from **TCGA-LUAD**, which represents **lung adenocarcinoma** samples. The model is designed to classify samples into:

- **Tumor**
- **Normal**
